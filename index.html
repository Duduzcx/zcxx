Sobre



Me chamo Eduardo, tenho 16 anos de idade. Desenvolvo páginas e sistemas web responsivos e de layout atrativo. Tenho uma trajetória de cerca de 2 anos, não sou profissional na aréa, porém, ja fiz alguns cursos e tenho algumas experiências, o que me está me tornando um profissional dedicado e organizado. Hoje na área de desenvolvimento vivo o aprendizado continuo, focando em tecnologias JavaScript, HTML, Css e Python. Busco oportunidade no mercado como dev e, em um futuro próximo, viso me tornar um desenvolvedor Full Stack. Sou apaixonado por tecnologia e games, mas principalmente em desenvolvimento. Estudo tecnologias de linguagens Back-end, para futuramente me tornar Full Stack. Também estou imerso nos estudos das tecnologias DevOps.




Conhecimentos




HTML5

HTML5 é uma linguagem de marcação de hipertexto, que usamos para marcar os elementos da página e construí-la.

CSS3

CSS3 é uma linguagem de estilo usada para descrever a apresentação dos elementos da página os estilizando.

JavaScript

JavaScript é uma linguagem de programação interpretada usada no desenvolvimento web para descrever funcionalidades e comportamentos da página.
